OmniHistoryPlus Options

These settings let you tweak how OmniHistoryPlus behaves.

Max results to retrieve from Chrome API:
Default is 200. The higher the value, the slower it might return results if there are a lot in your browse history for your search terms.

Max results to filter for display:
Default is 10. It controls how many of the results (from the Chrome API) are sent to the omnibar for display.

Show Debug Info:
Shows the visit count and last visit date for any results displayed in the omnibox.
